Initial release of JellyWatch: a Jellyfin plugin that mirrors the full media library to a GitHub-hosted catalog.json for off-network browsing.

## Plugin

- Supports movies, TV series, episodes, audio tracks, music albums, music artists, books, music videos and audiobooks.
- Auto-syncs on startup (2-minute delay) and every 7 days. No manual trigger exposed.
- Debounces library change events (default 30s quiet window) before pushing.
- Pushes to GitHub via the Contents API with a fine-grained PAT (Contents: read+write only).
- Plugin GUID: a12a453e-31f7-46e2-9968-d92580109f24. Namespace: Jellyfin.Plugin.JellyWatch.

## Viewer

- Static HTML+JS site served from GitHub Pages (docs/ folder).
- Shows 20 most recent items per media type on the main page.
- Global search field searches across all types simultaneously; Escape clears back to sections.
- Mobile-first layout, no CDN dependencies, no build step.
